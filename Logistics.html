About This Project

A mini logistics management website focusing on backend development, featuring secure user login authentication and basic stock management with real-time database integration.

  • User Authentication

    – Secure login system with database validation
  • Data Visualization

    – Dynamic charts to display key logistics data
  • Database Integration

    – Backend retrieves and displays data in real time
  • Responsive Interface

    – Simple yet responsive design for better usability
  • Backend: Node.js (Express)
  • Database: SQL Server
  • Frontend: HTML, CSS, JavaScript
