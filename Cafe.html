About This Project

A responsive café website built with HTML, CSS, and Bootstrap. It features a dynamic navigation menu, gallery grid layout, and a functional reservation form to simulate real customer interaction.

  • Responsive Design

    – Optimized layout for both desktop and mobile devices
  • Menu Display

    – Showcases café menu items with categories and details
  • User-Friendly Navigation

    – Clean and intuitive interface for visitors
  • Contact & Information Section

    – Includes essential café information for customers
  • Frontend:

    HTML, CSS, Bootstrap
